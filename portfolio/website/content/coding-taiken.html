CSSで絵を描く方法

今月のコンテンツでCSSを使って絵を描く体験をします。最近はデザイナーがコーディング基礎を学ぶことが重要になってきています。一方、ベクターアートを描くことがレイアウト、色や形態の使い方を身につけることができるのでCSSで絵を描くことはデザイナーのみならずフロントエンドエンジニアにも有益です。

コーディングを体験しましょう

では今回はコアラを描きます。下記の順番でテキストエディターに入力すると隣で結果が現れます。詳しく知りたい場合はエディターの下に開設しています。

01背色を付けて、キャンバスを用意する

下記の通りの文を打ち込むと、円が現れる。下記の通りの文を打ち込むと、円が現れる。コードの解説をする。 


                body{
                  background: #25A9FC;
                }

コアラの絵は透明な四角形体の上に描くので、この四角形体をHTMLのボディーの真ん中にする。コアラの頭はこの四角形体の中に配置される。 

                    
                        .box{
                          position: relative;
                          margin: auto;
                          display: block;
                          margin-top: 8%;
                          width: 600px;
                          height: 420px;
                          background: none;
                        }

02頭を書く

Webページの中に頭になる円形を描きます。

        
            .head{
              position: absolute;
              top: 16.5%;
              left: 25%;
              width: 50%;
              height: 67%;
              background: #A6BECF;
              border-radius: 50%;
            }

もう一つの円形を描こう。この円形は耳を頭のレイヤーの下から出ているように見えるために描く。このステップでは見た目が変わらない。 


  .head-copy{
    width: 100%;
    height: 100%;
    position: absolute;
    background: #A6BECF;
    border-radius: 50%;
    z-index: 2;
  }

03耳を描く

耳が2つの薄いグレーの大きい円形とちょっと濃いめグレーの小さい円形で作られ、頭の両側が一つずつある。 


  .ear-left{
    position: absolute;
    width: 60%;
    height: 65%;
    left: -20%;
    top: 5%;
    background: #A6BECF;
    border-radius: 50%;
    z-index: 1;
  }

      .ear-right{
        position: absolute;
        width: 60%;
        height: 65%;
        right: -20%;
        top: 5%;
        background: #A6BECF;
        border-radius: 50%;
        z-index: 1;
      }
  
        .inner-ear{
          position: absolute;
          border-radius: 50%;
          width: 80%;
          height: 80%;
          top: 10%;
          left: 10%;
          background: #819CAF;
        }

04目を描く

目の作り方も耳と似ている。二つの大きい白い円形と黒い瞳の小さい円形で作られている。 


      .eye-left{
        position: absolute;
        background: white;
        width: 30%;
        height: 33%;
        top: 25%;
        left: 21%;
        border-radius: 50%;
        z-index: 3;
      }

        .eye-right{
          position: absolute;
          background: white;
          width: 30%;
          height: 33%;
          top: 25%;
          right: 21%;
          border-radius: 50%;
          z-index: 2;
        }
    

      .pupil{
        position: absolute;
        width: 28%;
        height: 30%;
        top: 35%;
        left: 36%;
        border-radius: 50%;
        background: #27354A;
      }

05鼻を描く

鼻は頭の真ん中の少し下で配置する。z-index:4で目の上に置くことができる。鼻の形は最初に縦長の長方形を作り、そしてborder-radius:50pxで角を丸くさせる。 


        .nose{
          position: absolute;
          background: #BE845F;
          width: 25%;
          height: 42.5%;
          left: 37%;
          top: 45%;
          border-radius: 50px;
          z-index: 4;
        }
